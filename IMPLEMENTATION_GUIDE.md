# HOKORI ゲーム実装ガイド

## 概要

**HOKORI** は、ほこり視点のステルス成長ゲームです。プレイヤーは小さなほこりとなり、人間に見つからないように隠れながら、他のほこりを吸収して大きく成長することを目指します。

## ゲームの特徴

### 1. ステルス成長メカニクス
- **段階的な成長フェーズ** - 6段階（Tiny → Legendary）
- **隠密度スコア** - 0-100で隠れている度合いを表示
- **人間への検出確率** - 距離、明るさ、サイズで計算

### 2. ゲームプレイシステム
- **複合スコア計算** - 吸収、コンボ、ステルス、時間ボーナス
- **人間AI** - 学習メモリ、複数行動パターン
- **環境インタラクション** - 複数の隠れ場所、時間帯システム

### 3. デザイン・UI/UX
- **完全レスポンシブ** - スマホ縦横対応
- **ビジュアルエフェクト** - パーティクル、フローティングテキスト、フラッシュ
- **直感的なHUD** - 状態表示、スコア、ステルス度

### 4. ゲームコンテンツ
- **6つのステージ** - 寝室、リビング、キッチン、浴室、地下室、屋根裏
- **3つのボスステージ** - 掃除機マスター、人間ハンター、究極のチャレンジ
- **拡張オーディオ** - テンション連動BGM、複数の効果音

## 隠密度スコア計算

隠密度スコア（0-100）は以下の要因から計算されます：

```
隠密度 = 
  (1 - 明るさ) × 35% +
  (1 - 移動速度/200) × 25% +
  (1 - サイズ/20) × 20% +
  隠れ判定 × 20%
```

### ステルス状態
- **PERFECT** (90-100) - 完全に隠れている
- **EXCELLENT** (75-89) - 非常に隠れている
- **GOOD** (60-74) - よく隠れている
- **FAIR** (40-59) - まあまあ隠れている
- **POOR** (20-39) - あまり隠れていない
- **EXPOSED** (0-19) - 見つかる危険

## 人間AI

### 学習メモリ
人間は同じ場所で何度もプレイヤーを見つけると警戒度が上昇します。

- 1回目の検出 - 脅威度 1.0倍
- 2回目の検出 - 脅威度 1.3倍
- 3回目以上 - 脅威度 1.6倍

### 行動パターン
| パターン | 説明 | 条件 |
|---------|------|------|
| PATROL | 通常巡回 | 疑念値 < 0.2 |
| ALERT | 警戒状態 | 0.2 ≤ 疑念値 < 0.85 |
| CHASE | 追跡状態 | プレイヤーが見える |
| VACUUM | 掃除機起動 | 疑念値 ≥ 0.85 |

### 隠れ場所の種類

| 場所 | ステルス | 安全度 | アクセス | 最大滞在 |
|------|---------|--------|---------|---------|
| ベッド下 | 95% | 90% | 70% | 10秒 |
| クローゼット | 85% | 70% | 60% | 5秒 |
| 家具の隙間 | 75% | 50% | 80% | 3秒 |
| カーペット下 | 65% | 40% | 90% | 2秒 |

隠れ場所が何度も発見されると、ステルス効果が低下します。

- 1回発見 - 効果 100%
- 2回発見 - 効果 85%
- 3回発見 - 効果 70%

## スコアシステム

### スコア計算式

```
総スコア = 吸収スコア + コンボボーナス + ステルスボーナス + 時間ボーナス + 達成度ボーナス
```

### 各要素の計算

**吸収スコア**
```
基本スコア = サイズ × 10 + 吸収数 × 2
吸収スコア = 基本スコア × (1 + コンボ × 0.15) × 難易度倍率
```

**コンボボーナス**
- コンボ3-4 - 基本スコア × 10
- コンボ5-9 - 基本スコア × 20
- コンボ10+ - コンボ数² × 5

**ステルスボーナス**
```
ステルスボーナス = (隠密度スコア × 1.5 + 隠れ時間 × 15) × 難易度倍率
```

**時間ボーナス**
```
時間ボーナス = 残り時間 × 20 × 難易度倍率
```

**達成度ボーナス**
- 目標達成 - 500点
- 目標達成 + 20% - 800点
- サイズ15以上 - 1000点追加

### 難易度倍率
- Easy - 0.8倍
- Normal - 1.0倍
- Hard - 1.3倍
