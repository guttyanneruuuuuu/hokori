# DUST — ほこりサバイバル 改善計画書

## 1. コンセプトの再定義
「ほこり」の視点から見た、巨大で危険な「人間の世界」を生き抜くステルス・サバイバル。
単に大きくなるだけでなく、**「バレたら終わり」という緊張感**と、**「暗がりの安心感」**を強調する。

## 2. デザイン・ビジュアルの刷新
- **ダークで高品質な質感**: 
  - 背景にフローリングやカーペットのテクスチャを導入。
  - プレイヤー（ほこり）の質感をより「ふわふわ」させ、大きくなるにつれてゴミ（髪の毛、糸くず）が絡みつく表現を追加。
- **ライティングの強化**:
  - 人間の懐中電灯や部屋の照明による動的な影の表現をよりリアルに。
  - 「安全な暗がり」と「危険な光」のコントラストを視覚的に分かりやすく。

## 3. ゲーム性の向上
- **ステルス要素の深化**:
  - 「バレる」までの猶予（警戒ゲージ）を導入。
  - 人間の足音や掃除機の音を画面の振動や波紋エフェクトで視覚化。
- **成長システム**:
  - 食べるものによってステータスが変化（例：髪の毛を食べると移動が遅くなるが耐久力アップ）。
- **ステージギミック**:
  - ルンバ（掃除ロボット）の巡回、人間の突然の歩行、換気扇の風による強制移動。

## 4. スマホ対応・操作性の改善
- **レスポンシブ対応**:
  - 縦画面（Portrait）と横画面（Landscape）でUI配置を動的に変更。
  - 画面回転時にキャンバスサイズを最適化。
- **操作性**:
  - ダイナミックジョイスティック（画面のどこを触っても操作開始できる）を実装。
  - 触覚フィードバック（バイブレーション）の活用。

## 5. 実装フェーズ
1. **Phase 1: 基盤強化** (スマホ対応、レスポンシブ、操作性)
2. **Phase 2: ゲーム性拡張** (AI、ステルス、成長ロジック)
3. **Phase 3: ビジュアル刷新** (テクスチャ、エフェクト、UIデザイン)
4. **Phase 4: 最終調整** (バランス調整、サウンド、バグ修正)
