# DUST — ほこりサバイバル クオリティアップ計画

## 1. デザイン・ビジュアルの究極進化
- **「ぷにぷに」感とリアリティの融合**:
  - プレイヤー（ほこり）の描画をより有機的に。大きくなるにつれて絡みつく「ゴミ（髪の毛、糸くず）」をプロシージャルに描画。
  - 背景（床）に微細なテクスチャ（木目、畳、カーペット）を動的に生成して描画し、没入感を高める。
- **動的な環境表現**:
  - 換気扇や人間の移動による「風」をパーティクルで視覚化。
  - ライティングをさらに強化し、影の境界線を柔らかく、かつ「隠れている感」を演出。

## 2. ゲーム性の深化（ステルス＆アクション）
- **「バレない」緊張感の強化**:
  - 人間の「足音」を視覚的な波紋（サウンドリング）として表示。近づく恐怖を演出。
  - 警戒レベルに応じたBGMの動的変化（より緊迫したアレンジへ）。
- **特殊アクションとスキルの追加**:
  - **「静止」**: 動かないことでさらに見つかりにくくなる。
  - **「分裂」**: 危機を回避するためにサイズを犠牲にして小さく分かれる（要検討）。
- **新しい敵とギミック**:
  - **「ペット（猫・犬）」**: 人間とは違う予測不能な動きで追いかけてくる。
  - **「粘着トラップ（コロコロ）」**: 触れると動けなくなる危険地帯。

## 3. スマホ・操作性の完成度向上
- **完全レスポンシブ**:
  - 縦・横の切り替えをさらにスムーズに。UIの重なりを完全に排除。
  - ジョイスティックの感度設定や、ボタン配置の最適化。
- **ハプティクス（触覚）**:
  - 吸収した瞬間の「プチッ」とした感触や、危険時の心拍のような振動を細かく設定。

## 4. 追加要素
- **コレクション要素**:
  - 吸収した珍しいゴミの図鑑。
- **実績システム**:
  - 「一度も見つからずにクリア」「最大サイズ到達」などの称号。

## 実装ロードマップ
1. **[Phase 1] 操作性とレスポンシブの完成**: 縦横切り替えのバグ修正とジョイスティックの最適化。
2. **[Phase 2] ビジュアルアップデート**: 床のテクスチャとプレイヤーの描画強化。
3. **[Phase 3] ゲームコンテンツ強化**: 新しい敵（ペット）、ギミック、サウンドリングの実装。
4. **[Phase 4] UI/UXと最終調整**: 演出の強化、バランス調整、ハイスコア・実績の実装。
