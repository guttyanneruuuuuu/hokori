# 🌫️ DUST — ほこりサバイバル

> **あなたは、床の片隅で生まれた小さなホコリ。**
> 掃除機に吸われる前に、仲間のホコリを吸収して「大きな埃の王」を目指せ。

## 🎮 ゲーム概要

`DUST` は、ホコリ視点で繰り広げられる **ステルス・サバイバル・成長ゲーム** です。
人間に見つからないように暗い場所を移動し、床に落ちている髪の毛・食べカス・小さなホコリを吸収して大きくなっていきます。

## ✨ 主な特徴と改善点

- 📱 **モバイル最適化** — スマートフォンでの縦画面・横画面表示に完全対応し、直感的で快適なタッチ操作を実現。
- 🕹️ **操作性の向上** — ダイナミックジョイスティックと振動フィードバックにより、没入感のある操作体験を提供。
- 👻 **強化されたステルス要素** — 視覚的なフィードバックと隠密行動へのボーナスを導入し、戦略的なプレイを促進。
- 💡 **高品質なグラフィックとエフェクト** — パーティクルとライティングシステムを改善し、視覚的な魅力を向上。
- 🧠 **進化したAI** — 人間や掃除ロボットの行動パターンをより賢く、予測不能に調整。
- 🔊 **環境音演出** — 足音、掃除機、テレビ……人間の気配を音で感じる
- 🌪️ **成長メカニクス** — 大きくなるほど移動は重く、目立ちやすくなるリスク

## 🕹️ 操作方法

| 操作デバイス | 動作 |
|---|---|
| **PC** | |
| `WASD` / `矢印キー` | 移動 |
| `Shift`（押している間）| 静かに歩く（音が小さい） |
| `Space` | ダッシュ（音が大きく目立つ） |
| `P` / `Esc` | ポーズ |
| **スマートフォン** | |
| 🕹️ 左下のジョイスティック | 移動 |
| 🤫 SNEAK ボタン | 静かに歩く |
| ⚡ DASH ボタン | ダッシュ |
| 📱 縦持ち・横持ち | どちらでも快適にプレイ可能 |

## 🚀 起動方法

ブラウザで `index.html` を開くだけです。
```bash
# ローカルサーバーで起動する場合
python3 -m http.server 8000
# → http://localhost:8000
```

## 🛠️ 技術スタック

- Vanilla JavaScript (ES Modules)
- HTML5 Canvas 2D
- Web Audio API（プロシージャル SE）
- 依存ライブラリなし

## 🎯 開発目標

本プロジェクトでは、以下の目標を掲げ、ゲームの品質向上とユーザー体験の最大化を目指します。

1.  **スマホ操作性の向上とレスポンシブ対応の改善**: 既存のタッチコントロールをさらに洗練させ、縦横画面でのレイアウト調整を最適化します。ダイナミックジョイスティックの実装や、より自然な振動フィードバックの導入を検討します。
2.  **ステルス要素の強化とAIの調整**: プレイヤーの隠密行動に対する視覚的・聴覚的フィードバックを強化し、隠れることの重要性を高めます。人間や掃除ロボットのAIを改善し、より挑戦的でリアルな挙動を実現します。
3.  **グラフィックとエフェクトの高品質化**: パーティクルシステムとライティングシステムを詳細に調整し、視覚的な没入感を深めます。特に、ほこりの成長や吸収時のエフェクト、明暗の表現を強化します。
4.  **ゲームバランス調整と最終UIのブラッシュアップ**: 難易度設定の調整、パワーアップアイテムの効果見直し、スコアリングシステムの改善を行います。また、HUDやメニュー画面の視認性とデザインを向上させ、全体的なUI/UXを洗練させます。
5.  **バグ修正とパフォーマンス最適化**: 既存のバグを特定し修正するとともに、全プラットフォームでのスムーズな動作を目指し、パフォーマンスの最適化を行います。
