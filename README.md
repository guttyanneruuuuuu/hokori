# 🌫️ DUST — ほこりサバイバル

> **あなたは、床の片隅で生まれた小さなホコリ。**
> 掃除機に吸われる前に、仲間のホコリを吸収して「大きな埃の王」を目指せ。

## 🎮 ゲーム概要

`DUST` は、ホコリ視点で繰り広げられる **ステルス・サバイバル・成長ゲーム** です。
人間に見つからないように暗い場所を移動し、床に落ちている髪の毛・食べカス・小さなホコリを吸収して大きくなっていきます。

### 特徴
- 🕯️ **暗所ステルスシステム** — 光が当たる場所では発見されやすく、影に隠れると安全
- 👁️ **AI 視界システム** — 人間 / 掃除ロボットが視界コーンであなたを追う
- 🌪️ **成長メカニクス** — 大きくなるほど移動は重く、目立ちやすくなるリスク
- 🎨 **ダーク&可愛い** — ベクター描画によるパーティクル演出と柔らかな照明
- 🔊 **環境音演出** — 足音、掃除機、テレビ……人間の気配を音で感じる

## 🕹️ 操作方法

| キー | 動作 |
|---|---|
| `WASD` / `矢印キー` | 移動 |
| `Shift`（押している間）| 静かに歩く（音が小さい） |
| `Space` | ダッシュ（音が大きく目立つ） |
| `P` / `Esc` | ポーズ |

## 🚀 起動方法

ブラウザで `index.html` を開くだけです。
```bash
# ローカルサーバーで起動する場合
python3 -m http.server 8000
# → http://localhost:8000
```

## 🛠️ 技術

- Vanilla JavaScript (ES Modules)
- HTML5 Canvas 2D
- Web Audio API（プロシージャル SE）
- 依存ライブラリなし
